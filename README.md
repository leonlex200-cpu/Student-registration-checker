
# Student Registration Status Checker

## Purpose
This project shows how to check a student's registration status in four different programming languages.  
If a student registers more than 7 units, the program says **"Overload - Approval Required"**.  
Otherwise, it says **"Registration Accepted"**.

# Language Comparison

## Python
- Short and easy to read
- Great for beginners

## Java
- Longer code, very structured

## JavaScript
- Runs with Node.js or browsers
- Popular for web development

## C
- Oldest and most strict
- Very fast but harder for beginners

## Summary
- **Python** is easiest.  
- **Java** is strict but organized.  
- **JavaScript** is flexible.
